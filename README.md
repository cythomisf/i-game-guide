# i-game-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件的形式存在，不依赖后端服务，可直接在浏览器中打开浏览。

## 项目简介

i-game-guide 是一个以 HTML 页面为主要内容的归档仓库。所有页面均为独立文件，彼此之间没有强依赖关系。仓库不针对任何特定平台或网站，仅作为页面内容的结构化存储与展示。

## 目录说明

仓库根目录下包含多个 HTML 文件，部分页面可能存放在子目录中，以便按主题或用途进行简单分类。具体文件与目录结构如下：

```
i-game-guide/
├── index.html              # 仓库首页（可选）
├── page-a.html             # 示例页面 A
├── page-b.html             # 示例页面 B
├── docs/                   # 文档或说明类页面
│   └── example.html
└── archive/                # 已归档的旧版本页面
    └── old-page.html
```

各页面内容独立，目录结构仅用于组织管理，不影响页面功能。

## 页面归档说明

本仓库中的 HTML 页面主要来自以下来源：

- 自行编写的静态页面
- 整理收集的公开资料页面
- 测试或演示用途的独立页面

所有页面均以原始 HTML 格式存储，不进行混淆或加密。部分页面可能包含外部资源引用（如 CSS、JavaScript 库），这些资源均指向公开 CDN 或本地文件。

## 维护说明

- 本仓库不定期更新，新增或修改页面会通过提交记录体现。
- 不保证所有页面长期可用或兼容所有浏览器，但会尽量保持基础功能正常。
- 如发现页面内容错误或链接失效，欢迎通过 Issues 反馈，或直接提交 Pull Request 修正。
- 仓库维护者保留删除、修改或归档任意页面的权利。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开所需的 HTML 文件即可查看。
3. 也可通过 GitHub Pages 等服务部署后在线访问（需自行配置）。
